Voice AI Widget Demo

Welcome to the Voice AI Widget demonstration. This widget enables natural voice conversations with an AI assistant.

Features

  • Voice input using browser's Speech Recognition API
  • Text input as an alternative
  • AI responses powered by OpenRouter
  • Natural voice output via Eleven Labs
  • Conversation history
  • Compact, embeddable design

How to Use

Look for the blue chat button in the bottom-right corner. Click it to open the widget!

  1. Click the chat button to open the widget
  2. Click the microphone icon and speak, or type your message
  3. The AI will respond with text and voice
  4. Continue the conversation naturally

Browser Requirements

For voice input, use Chrome, Edge, or Safari. Text input works in all modern browsers.
